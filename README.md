# 🎭 Gapster

A free browser-based fill-in-the-blank word game with 120 hilarious stories, daily challenges, leaderboards and badges.

## 🎮 Play Now
👉 **[Gapster - Play Free](https://greathound67.github.io/Gapster)**

## ✨ Features
- 120 unique stories across themes like Adventure, Food, Animals, Sports and more
- Daily story that rotates every day
- Event stories for special holidays
- Google or email sign-in with full user profiles
- Leaderboards — most played, streaks, most badges,OG members
- Badge system with 35+ badges to earn
- Daily streak system
- Works well on both mobile and desktop

## 🛠️ Built With
- Vanilla HTML, CSS, JavaScript
- Supabase (auth + database)
- GitHub Pages (hosting)
